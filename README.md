# 微电网配置系统

一个基于 React + TypeScript + Vite 的微电网选型配置系统，帮助用户根据需求配置最适合的微电网解决方案。

🌐 **在线访问**: https://panskai.github.io/MicroGrid/

## 功能特性

### 🎯 核心功能
- **步骤式配置流程**：类似 semicrogrid.com 的交互式配置体验
- **三种配置场景**：
  - 已知负载情况：根据已知负载自动配置
  - 任意DIY：自由配置逆变器、电流等参数
  - 无负载情况：适用于新建项目或容量规划

### 📊 投资回报分析
- **投资回报曲线图**：可视化展示投资回报趋势
- **详细财务分析**：
  - 总投资成本
  - 年度节省金额
  - 投资回报年限
  - 20年累计收益

### 📋 选型报告
- **完整配置参数**：展示所有选型参数
- **系统容量分析**：光伏装机容量、储能容量等
- **成本分析**：详细的成本分解和收益预测

### 🏢 公司信息
- **VoltageEnergy 联系信息**：
  - 公司地址
  - 联系电话
  - 电子邮箱
  - 官方网站
  - 业务咨询

## 技术栈

- **React 19** - UI框架
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **Recharts** - 图表库
- **CSS3** - 样式设计

## 配置流程

1. **选择配置场景**：根据项目需求选择配置场景
2. **折叠支架配置**：选择支架数量和组件型号
3. **柴油发电机**：选择是否已有发电机
4. **电压等级**：选择系统电压等级
5. **特殊配置**（根据场景）：
   - DIY场景：配置逆变器数量和电流
   - 无负载场景：选择光储一体化托盘容量
6. **储能容量**：选择储能支撑天数
7. **EMS控制方式**：选择能源管理系统控制方式
8. **查看报告**：生成完整的选型报告和投资回报分析

## 投资回报计算

系统会根据以下因素计算投资回报：

- 折叠支架成本
- 逆变器成本
- 储能系统成本
- EMS系统成本
- 年度电费节省
- 维护成本节省
- EMS控制方式优化收益
